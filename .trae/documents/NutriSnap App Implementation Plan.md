# План разработки Android-приложения NutriSnap

## Технологический стек
- **Язык**: Kotlin
- **UI**: Jetpack Compose (Material 3)
- **ИИ**: Groq API (Llama 3.2 Vision)
- **База данных**: Room (для истории и избранного)
- **Камера**: CameraX
- **Сеть**: Retrofit + OkHttp
- **Архитектура**: MVVM + Clean Architecture

## Этапы реализации

### Этап 1: Инициализация и Дизайн
1. Создание структуры проекта и настройка `build.gradle`.
2. Реализация темы **Healthy Green** (светлая и темная версии).
3. Создание системы Onboarding-экранов.
4. Разработка главного экрана с современным дизайном.

### Этап 2: Интеграция Groq AI
1. Реализация сетевого слоя для запросов к Groq.
2. Создание системы обработки мультимодальных запросов (Фото + Текст).
3. Настройка промптов для получения ответов на русском языке с КБЖУ и советами.

### Этап 3: Ввод данных и История
1. Реализация функционала Камеры и выбора из Галереи.
2. Добавление текстового и голосового ввода (Speech-to-Text).
3. Создание локальной БД Room для хранения истории приемов пищи.
4. Реализация экранов просмотра, редактирования и удаления записей из истории.

### Этап 4: Продвинутые функции
1. Реализация экспорта истории в PDF/TXT.
2. Создание виджета для рабочего стола.
3. Добавление системы "умных подсказок" на основе частоты использования продуктов.

### Этап 5: Финализация
1. Полная локализация интерфейса.
2. Тестирование и исправление ошибок.
3. Оптимизация производительности.
