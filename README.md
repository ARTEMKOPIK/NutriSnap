# NutriSnap - AI Calorie Analyzer

Приложение для анализа калорий и макронутриентов (БЖУ) с использованием ИИ Groq (Llama 3.3 и Llama 4 Maverick).

## Как запустить проект

1. Установите **Android Studio**.
2. Склонируйте репозиторий.
3. Дождитесь синхронизации Gradle.
4. Введите свой API ключ Groq в файле `NutriSnapApp.kt` (или настройте через секреты, см. ниже).
5. Запустите приложение на эмуляторе или реальном устройстве.

## Автоматизация (CI/CD)

В проекте настроен **GitHub Actions**. Каждый раз, когда вы пушите код в ветку `main`, GitHub автоматически:
1. Проверяет код на ошибки.
2. Собирает APK-файл.
3. Создает новый **Release** в вашем репозитории и прикрепляет туда готовый APK.

## Как залить на GitHub (Инструкция)

1. Создайте новый пустой репозиторий на GitHub.
2. **Важно:** Откройте проект в **Android Studio** хотя бы один раз. Студия автоматически создаст файлы `gradlew` и папку `gradle/wrapper`, которые необходимы для работы авто-сборки на GitHub.
3. В терминале в папке проекта выполните:
   ```bash
   git init
   git add .
   git commit -m "Initial commit: NutriSnap App with CI/CD"
   git branch -M main
   git remote add origin https://github.com/ВАШ_ЛОГИН/ВАШ_РЕПОЗИТОРИЙ.git
   git push -u origin main
   ```

## Безопасность (API Ключи)

**Важно:** Сейчас ваш API ключ прописан в коде. Если вы делаете репозиторий публичным, рекомендуется:
1. Добавить ключ в **GitHub Secrets** (Settings -> Secrets and variables -> Actions) под именем `GROQ_API_KEY`.
2. Изменить код так, чтобы он читал ключ из системных переменных или `BuildConfig`.
